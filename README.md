## TL; DR

在這個專案中，我們打算使用 Python 自訂一個 `SimpleCalculator` 類別；這個類別將具有四個方法：兩數相加、兩數相減、兩數相乘以及兩數相除。透過這個專案，我們能夠暸解如何將四個方法綁定在一個類別之中。
