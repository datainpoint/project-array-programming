## TL; DR

在這個專案中，我們打算使用 Python 的 NumPy 套件體驗五個陣列程式設計元素：資料結構（Data structure）、索引（Indexing）、向量化（Vectorization）、廣播（Broadcasting）與化簡（Reduction）。透過這個專案，我們能夠具體暸解 NumPy 是如何實現陣列程式設計。
